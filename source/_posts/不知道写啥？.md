# 学啥写啥？
今天学了啥呢，怎么控制`hexo`生成一个新的文章，首先要通过`new`的命令：
```bash
hexo new <layout> "文章名"
```
这个命令会调用scaffolds下面的模板创建，创建**post**的layout为`post`，通过layout下的post.md控制，
可以在post.md中设置文章的标题、标签、分类等信息。例如：
>---
title: {{ title }}
date: {{ date }}
tags:
---
这个就在生成文章时自动填充。同时也可以通过命令控制，例如：
```bash
hexo new post "文章名" --tags="思考,随笔,生活"
```
可以控制的参数有`--author`、`--categories`、`--date`、`--tags`等。具体可以查阅[hexo文档](https://hexo.io/zh-cn/docs/front-matter)。

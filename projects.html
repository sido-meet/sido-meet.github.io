Error: Failed to fetch GitHub projects.

How to fix this:

  1. Make sure you have a GitHub account and have generated a personal access token.
  2. Create a .env file in the root directory of your Hexo project.
  3. Add your GitHub token to the .env file like this: GITHUB_TOKEN=your-token.
  4. Re-run hexo generate to fetch your GitHub projects.
